HTML stands for HyperText Markup Language.

  • HTML is the markup language we use to for describing the structure of Web pages.
  • HTML defines the elements displayed to the user, which are styled with CSS.
  • HTML has many tags that can be used to display media or text.

CSS stands for Cascading Style Sheets.

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.

JS stands for JavaScript

  • Javascript is the language we use to make a HTML page function & more responsive.
  • JavaScript is used to add functions like toggle hide/show of HTML elements.
  • JavaScript can detect window resize, change in HTML and much-much more.

PHP is an acronym for "PHP: Hypertext Preprocessor"

  • PHP can generate dynamic page content
  • PHP can create, open, read, write, delete, and close files on the server
  • PHP can collect form data

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

  • Python can be used on a server to create web applications.
  • Python can be used alongside software to create workflows.
  • Python can connect to database systems. It can also read and modify files.

Bootstrap is the most popular HTML, CSS, and JavaScript framework for developing responsive, mobile-first websites.

  • Bootstrap is a library that focuses on simplifying the development of informative web pages.
  • It has responsive structures & styles
  • It helps you design websites faster and easier.

React, sometimes referred to as a frontend JavaScript framework, is a JavaScript library created by Facebook.

  • React allows developers to create large web applications that can change data, without reloading the page.
  • The main purpose of React is to be fast, scalable, and simple.
  • It works only on user interfaces in the application.

Code.org is a non-profit organization and eponymous website led by Hadi and Ali Partovi that aims to encourage people, particularly school students in the United States, to learn computer science

  • Code.org uses the library p5.js
  • It enables us to do coding using blocks as well as by typing.
  • Code.org makes it wasy to write code for learning purposes.

p5.js is a JavaScript library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else! p5.js is free and open-source because we believe software, and the tools to learn it, should be accessible to everyone.

  • p5.js is used mostly for game development.
  • It makes it easy to use the HTML canvas element.
  • p5.js is optimized for game development.

Firebase is a platform developed by Google for creating mobile and web applications.

  • Firebase is a easy to use database.
  • Firebase is compatible with iOS apps, Android apps, Unity game engine and Web apps
  • Firebase has many functions that make it easy to post and get data.
